OVERVIEW
--------
Architecture Simulator adalah aplikasi web interaktif yang dirancang untuk 
solution architect, developer, dan tim IT untuk merancang, memvisualisasikan, 
dan mensimulasikan arsitektur perangkat lunak dengan sistem drag-and-drop 
yang intuitif.

File: architecture-simulator.html
Ukuran: ~6,842 baris kode
Teknologi: HTML5, CSS3, JavaScript ES6+, SVG, Font Awesome, Google Fonts

================================================================================
                              CARA PENGGUNAAN
================================================================================

1. MEMULAI:
   - Buka file architecture-simulator.html di browser
   - Klik "Tutorial" untuk panduan lengkap
   - Pilih quick template atau mulai dari scratch

2. MERANCANG ARSITEKTUR:
   - Drag komponen dari sidebar ke canvas
   - Hubungkan komponen dengan connection points
   - Atur posisi komponen sesuai kebutuhan

3. SIMULASI:
   - Klik tombol "Simulate" untuk menjalankan simulasi
   - Lihat data flow dan animasi perpindahan data
   - Monitor metrics dan performa real-time

4. PRESENTASI:
   - Gunakan mode "Present" untuk presentasi
   - Export arsitektur untuk dokumentasi
   - Simpan proyek untuk referensi masa depan
